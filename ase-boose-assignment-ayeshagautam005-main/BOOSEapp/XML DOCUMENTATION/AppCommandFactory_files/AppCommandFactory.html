Method CreateCommand
Creates an ICommand instance from the specified command line text.
Syntax
C#

Parameters
commandLine
The textual representation of the command, including any arguments, for example: "moveto 100 200" or "circle 50".
Return Value
An ICommand that corresponds to the parsed command text, or null if the command is not recognized or cannot be parsed.
Exceptions
Exception Condition
CanvasException Thrown when an error occurs while creating the command from the specified text.
