Class Program
The main entry point for the BOOSE Drawing Application. This class contains the static Main method that initializes and runs the Windows Forms application.
Syntax
C#
    static class Program

Method Main
The main entry point for the application. This method performs the following operations: 1. Enables visual styles for Windows Forms controls. 2. Sets the default text rendering compatibility mode. 3. Creates and runs the main application form.
Syntax
C#
    static void Main()
Remarks
The STAThreadAttribute is required for Windows Forms applications that use components like the Clipboard or drag-and-drop functionality.
