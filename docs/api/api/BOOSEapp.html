BOOSEapp Docs

Namespace BOOSEapp

Classes

AppArray

Represents a command that handles array creation requests, such as array int scores 10, within the BOOSE application.

AppCanvas

Represents a drawing surface that implements the ICanvas interface. Provides methods for drawing shapes, lines, and text.

AppCircle

Represents a command that draws a circle on a canvas.

AppCommandFactory

Provides a singleton factory for creating ICommand instances from text input.

AppDrawTo

Represents a command that draws a line from the current canvas position to the specified coordinates.

AppFor

Represents a command that handles parsing and reporting of FOR loop constructs.

AppIf

Represents a command that handles parsing and reporting of IF statement constructs.

AppInt

Represents a command that declares integer variables in the application context.

AppMethod

Represents a command that handles parsing and reporting of method definitions.

AppMoveTo

Represents a command that moves the pen to the specified coordinates on the canvas.

AppParser

Provides functionality to parse text lines into command objects using a command factory.

AppReal

Represents a command that declares real (floating-point) variables in the application context.

AppRectangle

Represents a command that draws a rectangle on the canvas using the specified dimensions.

AppWhile

Represents a command that handles parsing and reporting of WHILE loop constructs.

CanvasException

Represents errors that occur during canvas-related operations.

ClearCommand

Represents a command that clears the drawing canvas.

MainForm

Main form for the BOOSE Drawing Application. This form provides the user interface for entering drawing commands, executing them, and viewing the results on a drawing canvas.

MethodInfo

Represents a method definition, including its name, parameters, body, and return variable.

ResetCommand

Represents a command that resets the canvas state, including the pen position.

SetPenColorCommand

Represents a command that sets the current pen color on the canvas using RGB components.

WriteTextCommand

Represents a command that writes text onto the canvas at the current pen position.

Interfaces

ICanvas

Defines a contract for drawing operations on a canvas.

ICommand

Defines a command in the drawing program that follows the Command pattern to encapsulate a drawing operation.

ICommandParser

Defines a contract for parsing textual commands into executable command objects.
